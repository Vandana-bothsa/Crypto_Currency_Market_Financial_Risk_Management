# Machine Learning-Based Analysis of Crypto Currency Market Financial Risk Management

This project focuses on analyzing and managing financial risks in the cryptocurrency
market using machine learning techniques. The system helps in understanding market
trends and supports data-driven investment decisions.

## Project Description
Cryptocurrency markets are highly volatile and risky. This project uses machine
learning algorithms to analyze historical cryptocurrency data and assess financial
risk, helping users make informed decisions.

## Technologies Used
- Python
- Django
- Machine Learning (Scikit-learn)
- MySQL
- Pandas, NumPy
- Seaborn
- WAMP Server

## Features
- User registration and login
- Cryptocurrency data analysis
- Risk prediction using machine learning
- Graphical visualization of market trends
- Admin and user modules

## Dataset
The dataset contains historical cryptocurrency market data used for training and
testing the machine learning models.

## How to Run the Project

### Prerequisites
- Python 3.6.2
- WAMP Server
- PyCharm

### Installation Steps
1. Install required Python packages:
pip install django==3.2
pip install pandas numpy seaborn
pip install scikit-learn
pip install mysqlclient
pip install xlwt openpyxl


2. Start WAMP Server and open phpMyAdmin:
http://localhost/phpmyadmin/

3. Import the database.

4. Run Django migrations:
python manage.py makemigrations
python manage.py migrate

5. Start the server:
python manage.py runserver

6. Open browser:
http://127.0.0.1:8000
